# quantitative-analysis-finance-projects
Projet explorant les corrélations entre météo à New York et rendements d’actifs financiers (S&amp;P 500, VIX, gaz, utilities, maïs). L’analyse a ensuite évolué vers le maïs en intégrant la météo des zones de production américaines afin d’évaluer son lien potentiel avec les variations de rendement, sans établir de causalité.
